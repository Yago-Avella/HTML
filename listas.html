Para añadir las listas ordenadas tienes que añadir la etiqueta ol y en cada "fila" de la lista poner la etiqueta li, al final de la lista, in troduce un /ol

Para añadir las listas desordenadas tienes que añadir la etiqueta ul y en cada "fila" de la lista poner la etiqueta li, al final de la lista, introduce un /ul

Para mezclar las listas tendrás que empezar con el orden que prefieras, ol para ordenadas y ul para desordenadas y dentro de esta etiqueta insertar la lista del contrario elegido anteriormente, una vez insertado el li del primer numero, puedes introducir las sublistas con la etiqueta de listas que prefieras dentro de esta, cada fila necesitará tener lka etiqueta li también
